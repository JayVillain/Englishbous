Modul 13: Conditional Sentences

Conditional Sentences (kalimat pengandaian) digunakan untuk menyatakan situasi hipotetis atau tidak nyata dan konsekuensinya. Ada beberapa jenis Conditional Sentences, masing-masing dengan struktur dan penggunaan yang berbeda.

1. Conditional Type 0 (General Truths / Kebenaran Umum)

Digunakan untuk fakta umum, kebiasaan, atau kebenaran ilmiah yang selalu benar.

Struktur:

If + Simple Present, Simple Present

  • If you heat ice, it melts. (Jika kamu memanaskan es, itu meleleh.)
  • If it rains, the ground gets wet. (Jika hujan, tanah menjadi basah.)

2. Conditional Type 1 (Real or Likely Situations / Situasi Nyata atau Mungkin)

Digunakan untuk menyatakan situasi yang mungkin terjadi di masa depan dan konsekuensinya.

Struktur:

If + Simple Present, Will + Base Form (infinitive)

  • If it rains tomorrow, I will stay at home. (Jika besok hujan, saya akan tinggal di rumah.)
  • If she studies hard, she will pass the exam. (Jika dia belajar keras, dia akan lulus ujian.)

3. Conditional Type 2 (Unreal Present/Future / Tidak Nyata Saat Ini/Masa Depan)

Digunakan untuk situasi hipotetis atau tidak nyata di masa sekarang atau masa depan, serta nasihat.

Struktur:

If + Simple Past, Would + Base Form (infinitive)

  • If I were rich, I would buy a big house. (Jika saya kaya, saya akan membeli rumah besar.) - (Saya tidak kaya sekarang)
  • If I had more time, I would read more books. (Jika saya punya lebih banyak waktu, saya akan membaca lebih banyak buku.)
  • If I were you, I would apologize. (Jika saya adalah kamu, saya akan meminta maaf.) - (Nasihat)

4. Conditional Type 3 (Unreal Past / Tidak Nyata di Masa Lalu)

Digunakan untuk menyatakan situasi di masa lalu yang tidak terjadi, dan konsekuensi hipotetisnya di masa lalu.

Struktur:

If + Past Perfect, Would have + Past Participle (V3)

  • If I had studied harder, I would have passed the exam. (Jika saya belajar lebih keras, saya pasti sudah lulus ujian.) - (Saya tidak belajar keras di masa lalu, jadi saya tidak lulus.)
  • If she had known about the party, she would have come. (Jika dia tahu tentang pesta itu, dia pasti datang.)
